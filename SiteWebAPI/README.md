# SiteWebAPI

API Node/Express para o projeto.

- Porta padrão: `5000`
- Rotas iniciais:
  - `GET /api/health` — status e timestamp

Scripts:
- `npm run dev` — desenvolvimento com recarga automática
- `npm start` — execução padrão

Configuração:
- Crie `.env` com:
  - `PORT=5000`
  - `CLIENT_URL=http://localhost:5173`
